Important HTML Tags

Various Heading tags

There are 6 Heading tags, h1 h2 h3 h4 h5 h6

H1 is the biggest Heading

H6 is the smallest Heading

Syntax:

<h1> </h1>

<h2> </h2>

<h6> </h6>

Output:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
heading 6

Paragraph Tag

It is used for adding content to the HTML Document.

Syntax:

<p> Statment........ </p>

Output:

This is a Paragraph.

Anchor Tag

It is used to link from one website to another website

Syntax:

<a href = "URL to website...."> Link </a>

Output:

This a link to google.com

Note:

If you click on the above link this will open in the same page.
If you want to open the link in different page then use Target attribute with "_blank" in <a> tag.

Syntax:

<a href = "URL to website...." target = "_blank" > Link </a>

Output:

This is a link to google.com which opens in new tab

Image Tag

It is used for adding Images into HTML Page.
It has two required attributes, they are:

  • src: This is the source from where image is being fetched.
  • alt: when image doesn't loads, then an alternative text will be displayed.

Syntax:

<img src = "URL of image" alt = "Alternative text when image doesn't load">

List Tag

There are two types of lists in HTML, they are :

  1. Unordered List.
  2. Ordered List.

Unordered List:

Unordered List is something, that we are representing a list in a bulleted format. In this we will use <ul> tag for start and <li> tag for each element.

Syntax:

<ul>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ul>

Output:

  • The First Item.
  • The Second Item.
  • The Third Item.

Ordered List:

Ordered List is something, that we represent a list in a numbered format. In this we will use <ol> tag for start and <li> tag for each element.
Commonly used attributes are start and type.

Syntax:

<ol>
    <li> Item 1 </li>
    <li> Item 2 </li>
    <li> Item 3 </li>
</ol>

Output:

  1. The First Item.
  2. The Second Item.
  3. The Third Item.

Note:

Nested list is also possible for both Ordered list and Unordered list.
